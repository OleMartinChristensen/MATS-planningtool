OPT._CheckConfigFile.Core.CheckConfigFile()[source]¶

Core function of CheckConfigFile.

Checks the values given in the Configuration File set by Set_ConfigFile and raises an error if any settings are found to be incompatible. Also prints out the currently selected Configuration File and which starting date and TLE it currently uses.
