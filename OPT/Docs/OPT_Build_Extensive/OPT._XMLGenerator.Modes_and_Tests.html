OPT._XMLGenerator.Modes_and_Tests package¶

This package consist of modules which contain functions that are called exclusively from the XML_generator_select function.

The functions are separated into modules depending on if they define a Mode, a Test, or a separate CMD. Depending on their definition, the functions either call for Macros and/or CMDs from the Macros_Commands package.

OPT._XMLGenerator.Modes_and_Tests.MODES module¶

Contain Mode functions which generates and calculates parameters for each mode, then calls for Macros/Commands functions located in the Macros_Commands package, which will in turn write commands to the XML tree object root.

Functions on the form "X", where X is any Mode:

Arguments:

root (lxml.etree.Element): XML tree structure. Main container object for the ElementTree API.

date (ephem.Date): Starting date of the Mode. On the form of the ephem.Date class.

duration (int): The duration of the mode [s].

relativeTime (int): The starting time of the mode with regard to the start of the timeline [s].

params (dict): Dictionary containing the settings of the Mode given in the Science_Mode_Timeline.

Returns:

None

When creating new Mode functions it is crucial that the function name is X, where X is the same as the string used in the Science Mode Timeline. Because OPT will automatically look for any function with the same name as the string in the Science Mode Timeline

OPT._XMLGenerator.Modes_and_Tests.SeparateCmds module¶

Calls CMDs, which will generate commands in the XML-file.

For PWRTOGGLE, PM, CCDBadColumn, CCDFlushBadColumns: Compares parameters given in the Science Mode Timeline to default parameters given in the set Configuration File and fills in any parameters missing in the Science Mode Timeline.

For the rest CMDS, each parameter must be given in the Science Mode Timeline.

PWRTOGGLE will always call a PWRTOGGLE_macro which also turns off all CCDs before power toggling.

Functions on the form "X", where X is any CMD:
Arguments:

root (lxml.etree.Element): XML tree structure. Main container object for the ElementTree API. lxml.etree.Element class

date (ephem.Date): Starting date of the CMD. On the form of the ephem.Date class.

duration (int): The duration of the CMD [s] as an integer class.

relativeTime (int): The starting time [s] of the CMD with regard to the start of the timeline as an integer class

params (dict): Dictionary containing the parameters of the CMD given in the Science_Mode_Timeline.

Returns:

None

@author: David
