OPT._TimelineAnalyzer package¶

The Timeline_analyzer part of the Operational_Planning_Tool, which purpose is to analyse a Science Mode Timeline and find out what Mode was running at given date.

Submodules¶

OPT._TimelineAnalyzer.Core module¶

Searches a Science Mode Timeline .json file for a given date and returns the scheduled mode and its settings. A part of the Operational Planning Tool.

OPT._TimelineAnalyzer.Core.Timeline_analyzer(science_mode_timeline_path, date)[source]¶

The core function of the Timeline_analyse program.

Parameters

  • science_mode_timeline_path (str) -- path to the .json file containing the Science Mode Timeline

  • date (str) -- A given date and time ('2019/09/05 12:09:25')

Returns

tuple containing:

Mode (str): The Mode scheduled at the given date. Settings (dict): The settings of the Mode.

Return type

(tuple)
