Operational_Planning_Tool._TimelineAnalyzer.Core module¶

Searches a Science Mode Timeline json file for a given date and returns the scheduled mode and its parameter. As part of the Operational Planning Tool.

Operational_Planning_Tool._TimelineAnalyzer.Core.Timeline_analyzer(science_mode_timeline_path, date)[source]¶

The core function of the Timeline_analyse program.

Parameters

  • science_mode_timeline_path (str) -- path to the .json file containing the Science Mode Timeline

  • date (str) -- A given date and time ('2019/09/05 12:09:25')

Output:

Mode (str): The currently scheduled Mode. Parameters (dict): The parameters of the Mode.
