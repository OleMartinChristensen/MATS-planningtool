Simulates MATS and the LP with or without yaw movement to be able to predict and schedule commands in the XML-file. Look at fixed limb altitude in operational mode. Operational_Planning_Tool._XMLGenerator.Modes_and_Tests.SeparateCmds module¶

Calls for macros, which will generate commands in the XML-file.

For PWRTOGGLE, PM, CCDBadColumn, CCDFlushBadColumns: Compares parameters given in the Science Mode Timeline to default parameters given in the set Configuration File and fills in any parameters missing in the Science Mode Timeline.

For the rest CMDS, each parameter must be given in the Science Mode Timeline.

Functions on the form "XML_generator_X", where the last X is any CMD:
Arguments:

root (lxml.etree.Element): XML tree structure. Main container object for the ElementTree API. lxml.etree.Element class

date (ephem.Date): Starting date of the CMD. On the form of the ephem.Date class.

duration (int): The duration of the CMD [s] as an integer class.

relativeTime (int): The starting time [s] of the CMD with regard to the start of the timeline as an integer class

params (dict): Dictionary containing the parameters of the CMD given in the Science_Mode_Timeline.

Returns:

None
