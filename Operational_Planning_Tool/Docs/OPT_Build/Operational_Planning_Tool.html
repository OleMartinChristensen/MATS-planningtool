Operational_Planning_Tool package¶

Contains the functions that can be called to invoke Timeline generator, XML generator, Data_Plotter and Timeline_analyse, which together constitute the Operational_Planning_Tool.

Operational_Planning_Tool uses a special .py file as a Configuration File.

Create a separate copy of the _ConfigFile and modify the settings in this copy.

A specific Configuration File and its date must be chosen using *Set_ConfigFile.

A timeline can be generated by using Timeline_gen. Timeline_gen schedules Modes and CMDs and saves it into a .json file.

This .json file can be converted with XML_gen into an .XML file, as described in the InnSat Payload Timeline XML Definition document, which will contain all the relevant commands to be sent to mission control.

Check_ConfigFile can be used to check if the values stated in the chosen Configuration File are plausible.

Examples

import Operational_Planning_Tool as OPT

OPT.Copy_ConfigFile('OPT_Config_File')

OPT.Set_ConfigFile('OPT_Config_File', '2019/09/05 08:00:00')

OPT.CheckConfigFile()

OPT.Timeline_gen()

OPT.XML_gen('Output/Science_Mode_Timeline__OPT_Config_File.json')

Settings for the programs are stated in a .py Configuration File, and the Configuration File to be used is set by calling Set_ConfigFile. The file must be visible in sys.path. A .py Configuration File with default values can be created by calling the Create_ConfigFile function. It is an copy of the original _ConfigFile, located in the Operational_Planning_Tool directory.

All settings in the generated Configuration File can be changed to adjust the operation of the Operational Planning Tool. All generated output files are saved in a folder called 'Output' in the working directory. Generated logs are also saved in folders created in the working directory.

Operational_Planning_Tool.Data_Plotter()[source]¶

Invokes the Data_Plotter program part of Operational_Planning_Tool.

Simulates the position and attitude of MATS during normal operation and compares it to the actual positional and attitude data given in a data set. Plots both the simulated data and given data.

Operational_Planning_Tool.Timeline_gen()[source]¶

Invokes the Timeline generator part of Operational Planning Tool.

Predicts and schedueles Science Modes and some separate payload CMDs into a list containing dates for each Mode and saves it to a .json file in /Output.

The Modes and CMDs to be scheduled are listed in OPT_Config_File.Modes_priority.

Settings for the operation of the program are stated in the Configuration File set by Set_ConfigFile.

Returns

None
