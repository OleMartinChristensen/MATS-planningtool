Operational_Planning_Tool._TimelineGenerator.Modes package¶

Contains modules which correspond to the scheduling of each Mode.

The top or core function of each module is the only one called from higher levels. The name of the core function in each Mode specific module must be equal to the name of the Mode as stated in OPT_Config_File.Modes_priority().

The core function may call two other functions inside the same module, date_calculator() and date_select. date_calculator returns calculated or chosen dates (1 or many). date_select chooses the most appropriate date or/and makes sure the date chosen is available.

Core function for the scheduling of a Mode.

Arguments:

Occupied_Timeline (dict of list): Dictionary with keys equal to planned and scheduled Modes with entries equal to their start and end time as a list.

Returns:

(dict of list): Occupied_Timeline (updated for the current Mode).

(str): Comment regarding the scheduling result of the active mode.

Operational_Planning_Tool._TimelineGenerator.Modes.Mode120.Mode120_date_select(Occupied_Timeline, dates)[source]¶

Subfunction, Either schedules a user provided date or a simulated date.

If automatic in OPT_Config_File is set to False, the date is user provided. It will be postponed until available.

If automatic in OPT_Config_File is set to True. A list of dictionaries containing simulated dates is provided. A date is selected for which the brightest star is visible at the minimum amount of H-offset in the FOV. If the date is occupied the same star will be selected with the 2nd least amount of H-offset and so on. Another star will not be chosen and if no date is available for the brightest star; the Mode will not be scheduled.

Parameters

  • Occupied_Timeline (dict of list) -- Dictionary with keys equal to planned and scheduled Modes together with their start and end time in a list. The list is empty if the Mode is unscheduled.

  • dates (ephem.Date) -- A list containing dictionaries containing parameters for each time a star is spotted.

  • dates -- A user provided date for the to schedule the Mode.

Returns

tuple containing:

(dict of list): Occupied_Timeline (updated with the result from the scheduled Mode).

(str): Comment regarding the result of scheduling of the mode.

Return type

Operational_Planning_Tool._TimelineGenerator.Modes.Mode200.Mode200_date_calculator()[source]¶

Subfunction, Either selects a user provided date, or simulates MATS FOV and stars.

Determines when the Moon is entering the FOV at an vertical offset-angle equal to #V-offset and also being located at a horizontal off-set angle equal to less than #H-offset when pointing at an LP altitude equal to #pointing_altitude.

A timeskip equal to 1/2 an orbit of MATS is applied after a successful spotting of the Moon.

Timeskips equal to the time it takes for the Moon orbital position to change by #H-offset degrees are also applied if the Moon is determined to be at an horizontal off-set angle larger then the horizontal FOV of the instrument, equal to #HFOV.

(# as defined in OPT_Config_File).

Saves the date and parameters regarding the spotting of the Moon Also saves relevant data to an .csv file located in Output/.

Arguments:

Returns

A list containing dictionaries containing parameters for each time the Moon is spotted.

Return type

dates ((list of dict))
